# 🍎 Calorie Fitness Tracker

A modern Android calorie tracking app built with **HTML, CSS, JavaScript, and Capacitor**.  
Track your meals, calories, protein, carbs, fat intake, and food photos in a clean and mobile-friendly interface.

---

## ✨ Features

- 📊 Track daily calories and nutrition
- 🥩 Monitor protein, carbs, and fat intake
- 📷 Upload or capture meal photos
- 🗂 Save daily meal history
- 🎯 Set custom nutrition goals
- 📱 Android-ready using Capacitor
- 💾 Local storage support (data saved on device)
- 🎨 Clean and responsive UI


---

## 🛠 Tech Stack

- HTML5
- CSS3
- JavaScript

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/yourusername/calorie-tracker-app.git
cd calorie-tracker-app
```

### 2. Install dependencies

```bash
npm install
```

### 3. Sync Capacitor

```bash
npx cap sync android
```

### 4. Open in Android Studio

```bash
npx cap open android
```

---

## 📱 Building the APK

1. Open the project in Android Studio
2. Let Gradle sync completely
3. Click:

```text
Build → Build APK(s)
```

The APK will be generated inside:

```text
android/app/build/outputs/apk/
```
